SSRSpeed需要Python环境支持,此教程为Windows下的操作步骤，不想装Python的，请到页面底部使用stairspeedtest_reborn

这是一个批量测试软件，非代理工具

不要无意义浪费流量


0. 请确保自己可以满足下列条件

(1) 已经下载了SSRSpeed

(1) 已经安装了Python

可选：如有V2测试需求，请下载v2ray-core并解压到clients文件夹

(2) 拥有任意可使用的节点

(3) 能够使用 Google 学会不明白的操作


1. 安装

(1) Python环境安装。注意！下载时32位64位不要搞混，安装时请勾选“ADD PYTHONx.x TO PATH”

(2) SSRSpeed下载解压

(3) 在解压目录按住"Shift"+"鼠标右键"打开命令提示符或PowerShell，复制粘贴以下代码，等待下载安装完成

python -m pip install --upgrade pip
pip3 install -r requirements.txt
pip3 install pillow
pip3 install requests
pip3 install pysocks

2. 开始测速

(1) 命令示例：

python main.py -u 订阅链接

(2) 其他命令自行查阅项目地址README
