SSRSpeed教程（Windows）


此测速结果仅供参考,由于网络环境的复杂多变,其不具备可靠性,仅代表此时此地此网络条件下的结果

此测试将会大幅消耗流量,请谨慎使用

请勿不经节点所有者同意擅自公开节点具体信息

此程序需要Python环境支持,此教程为Windows下的操作步骤


0. 请确保自己可以满足下列条件

(1) 已经下载了SSRSpeed

(1) 已经安装了Python

(2) 拥有任意可使用的节点

(3) 能够使用 Google 学会不明白的操作


1. 安装

(1) Python环境安装。注意！下载时32位64位不要搞混！

(2) SSRSpeed下载解压

(3) 在解压目录按住"Shift"+"鼠标右键"打开命令提示符或PowerShell，复制粘贴以下代码，等待下载安装完成

python -m pip install --upgrade pip
pip3 install -r requirements.txt
pip3 install pillow
pip3 install requests
pip3 install pysocks

2. 开始测速

(1) 命令示例：

python main.py -u 订阅链接

(2) 其他命令自行查阅项目地址README
