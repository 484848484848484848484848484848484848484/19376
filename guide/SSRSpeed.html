SSRSpeed需要Python环境支持,此教程为Windows下的操作步骤，不想装Python的，请到页面底部使用stairspeedtest_reborn


0. 请确保自己可以满足下列条件

(1) 已经下载了SSRSpeed

(1) 已经安装了Python

可选：如有V2测试需求，请下载v2ray-windows-64.zip并解压到clients文件夹

(2) 拥有任意可使用的节点

(3) 能够使用 Google 学会不明白的操作


1. 安装

(1) Python环境安装。注意！下载时32位64位不要搞混！

(2) SSRSpeed下载解压

(3) 在解压目录按住"Shift"+"鼠标右键"打开命令提示符或PowerShell，复制粘贴以下代码，等待下载安装完成

python -m pip install --upgrade pip
pip3 install -r requirements.txt
pip3 install pillow
pip3 install requests
pip3 install pysocks

2. 开始测速

(1) 命令示例：

python main.py -u 订阅链接

(2) 其他命令自行查阅项目地址README


不想输入命令？记不住命令？那就点击下面的按钮看看!

不想输入命令？试试这个批处理
或者复制下面框内代码保存为bat文件 
@echo off&
echo ##### 请放在ssrspeed目录下运行 #####
echo.
:start
echo 1：开始测速
echo 2：安装pip
echo 3：Web UI
echo 请选择（1/2/3）：
choice /c 123
if %errorlevel%==3 (goto :web)
if %errorlevel%==2 (goto :pip)
if %errorlevel%==1 (goto :test)

:pip
python -m pip install --upgrade pip
pip3 install -r requirements.txt
pip3 install pillow
pip3 install requests
pip3 install pysocks
pause
&& goto :start

:test
set /p a=请输入您的订阅链接:
if "%a%"=="" (
goto :test
) else (
python main.py -u "%a%"
)
pause
EXIT

:web
python web.py
